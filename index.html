Validação de Números com JavaScript

Instruções:

  1. Digite um número no campo "A"
  2. Digite um número no campo "B"
  3. Clique no botão "Avaliar Números" e o sistema irá dizer qual é o maior (verde) e menor (vermelho)

A validação dos números não acontece automáticamente, você precisa apertar o botão para validação

Caso queira começar outra validação aperte o botão "Limpa Formulario"
